# AI 545 - Artificial Intelligence

This repository contains course contents, and assignments for **AI 545: Artificial Intelligence**.

## Course Overview

Machine Learning Operations (ML Ops) focuses on the deployment, monitoring, and management of machine learning models in
production environments. This course covers end-to-end model lifecycle operations, including CI/CD pipelines,
versioning, model governance, and infrastructure automation for scalable AI deployment.
